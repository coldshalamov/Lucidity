### 2. GitHub Issues

For bug reports and feature requests:

**URL**: [github.com/lucidity-app/lucidity/issues](https://github.com/lucidity-app/lucidity/issues)

#### Reporting a Bug

Please include:
1. Device/OS (e.g., iPhone 14, iOS 17.2)
2. App version (check Settings)
3. Desktop OS (e.g., Windows 11)
4. Steps to reproduce
5. Expected vs actual behavior
6. Screenshots or logs (if available)

Use the **Bug Report** template when creating an issue.

#### Requesting a Feature

1. Search existing issues first
2. Use the **Feature Request** template
3. Describe the use case and value

## Security Vulnerabilities

If you discover a security vulnerability, please **do not** open a public issue.

Instead, email **security@lucidity.app** with:
1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Any suggested fixes

We will acknowledge within 24 hours and work with you on a fix.
